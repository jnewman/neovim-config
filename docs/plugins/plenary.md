# plenary-nvim

## Purpose

A general-purpose Lua utility library for Neovim plugins. Required as a dependency by telescope.nvim and octo.nvim; provides async, coroutine, path, and testing utilities.

## Keybindings

None — library dependency, no direct user-facing features.

## Config Notes

No configuration — installed purely to satisfy telescope.nvim and octo.nvim dependencies.
