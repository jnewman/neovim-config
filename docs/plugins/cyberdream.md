# cyberdream-nvim

## Purpose

A neon cyberpunk colorscheme with high-contrast colors. Included as an alternative to Catppuccin and Tokyo Night in the theme cycling rotation.

## Keybindings

| Key | Mode | Action |
|-----|------|--------|
| `<leader>tt` | Normal | Cycle to this theme (third in rotation) |

## Config Notes

- `italic_comments = true` — comments rendered in italics
- `borderless_telescope = false` — keeps telescope borders visible for clarity
- Activated via the theme cycler in `colorscheme.lua`; not set as the default
